# **App Name**: Kagozie Igbokwe

## Core Features:

- Terminal Hero: Display a dynamic terminal-style hero section with a JavaScript typewriter effect, including a blinking cursor.
- About Me Specs: Present a concise 'About Me' section featuring a tech stack grid of skills, rendered as styled chips.
- Projects Showcase: Implement a CSS Grid layout to display project cards with hover-activated 'View Code' button (GitHub icon).
- Contact Footer: Design a minimalist footer with large icons for GitHub, LinkedIn, and Email.

## Style Guidelines:

- Background: Deep charcoal/black (#121212) for a dark mode theme, offering high contrast and a professional look.
- Primary: Electric blue (#7DF9FF) for a terminal style accent, highlighting key elements.
- Accent: Soft grey (#EEEEEE) for text to ensure readability against the dark background.
- Header font: 'JetBrains Mono', a monospace font to simulate a code editor aesthetic.
- Body font: 'Inter', a clean sans-serif font for improved readability of body text and labels.
- FontAwesome icons for project links and social media, providing a clean and recognizable visual language.
- CSS Grid for project display, ensuring responsive behavior and adaptability across different screen sizes.
- Subtle hover animations on project cards, providing a gentle lift and reveal of the 'View Code' button to enhance interactivity.
